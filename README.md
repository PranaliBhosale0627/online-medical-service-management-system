
# 🏥 Online Medical Service Management System

## 📌 Project Overview
A full-stack Online Medical Service Management System built as a final year project. It provides features like patient management, and medical record handling using modern web technologies.The Online Medical Service Management System is a web-based application designed to simplify and digitize the process of medicine ordering, prescription handling, and delivery tracking.

Patients can browse medicines, upload prescriptions, place orders, and track delivery status. Admin manages medicines, users, prescription verification, and order processing.

Currently, the system supports Cash on Delivery (COD), with future plans to integrate online payment methods.

---

## 🚀 Key Features

### 👤 Patient Features
- User registration and login
- Browse available medicines
- Upload prescription for medicine approval
- Place medicine orders
- Track order status (Processing / Shipped / Delivered)
- Cash on Delivery (COD)

---

### 🏪 Admin Features
- Add, update, and delete medicines
- Manage medicine stock availability
- View and manage patient orders
- Verify uploaded prescriptions before approval
- Update order and delivery status
- Manage registered users

---

### 🌐 System Features
- Medicine ordering system
- Prescription-based approval workflow
- Order tracking system
- Secure login authentication
- Centralized medical service management
- Scalable for future enhancements

---
